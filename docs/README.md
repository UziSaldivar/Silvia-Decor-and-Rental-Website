# Project Title
Silvia Party Decor and Rental Business Website
## Description
For this project, I created a business website for a family member that has a small start up business related to Party rental and decorations. 

This website aims to solve her problems related to:
- Explains the type of service she offers
- Showcase her past work to the public
- Shows prices for items she has to rent
- Provides a method for clients to submit inquires requesting service

## Target Browser 
Currently, works well on any computer browser (Chrome, Firefox, Safari, etc) 

Eventually, it will support mobile devices in the near future.
